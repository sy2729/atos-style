# Color Schema

# Brand/Primary color

Atos blue is the brand color to be used across our product to display our charateristics and communicate ideas. It serves as the primary color of the design and should have a series of variation around it in different status.

#0066A1 Default Status Atos blue-1
#66A3C7 Hover Status Atos blue-2
#0043A1 Selected Status Atos blue-3

# Functional Color

Functional color conveys clear message to users regarding their actions or hinting the actions. Those colors should comply with the user's basic understanding of colors.

#01A050 Success
#E0111B Failure
#FFDC64 Alert
#0065A1 Link/Reminder

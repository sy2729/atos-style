# Font

# Font family

We use Stag and Proxima Nova as our font family across different digital designs. It also varies within the font family according to the content and usage.

# Font Scale

Our base font size is 14 for mobile on iOS and Android, and the corresponding line spacing is 22. Our base font size is 14 for web and the corresponding line spacing is 26.

Mobile

Type Font Size
title Stag Sans Book 24
subtitle Stag Sans Book 18
body text Stag sans book 16
minor text Stag sans light 12

Desktop

Type Font Size
title Stag Sans Medium 36px
subtitle Stag Sans Light 21px
body text Stag Sans Book 16px
minor text Stag Sans Light 14px
